## Sobre Mim

Sou focado em criar automacoes e sistemas inteligentes que resolvem problemas reais de empresas.

Meu trabalho une IA, n8n, APIs, bancos de dados, chatbots e aplicativos para transformar processos manuais em fluxos automatizados, organizados e escalaveis.

Atualmente estou desenvolvendo solucoes para:

- Atendimento automatizado com IA
- Chatbots conectados a sistemas reais
- Automacoes comerciais e operacionais
- Agendamento, pagamentos e notificacoes
- Apps mobile com FlutterFlow
- Backends com Supabase
- Integracoes via APIs, webhooks e bancos de dados

## Minha Forma De Trabalhar

Gosto de criar sistemas que uma empresa consegue usar de verdade no dia a dia.

Para mim, uma boa automacao precisa ser clara, confiavel e facil de manter. Nao basta conectar ferramentas: o fluxo precisa fazer sentido para o negocio, reduzir trabalho manual e gerar resultado pratico.
